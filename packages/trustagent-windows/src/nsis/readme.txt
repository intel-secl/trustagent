Silent Installation Commandline syntax: Setup_trustagent.exe /E="%~trustagent.env~%" /S


System Tray Icon Options:
1. Start - Starts Intel TrustAgent Service
2. Stop - Stops Intel TrustAgent Service
3. Setup - Runs "%TRUSTAGENT_HOME%\\bin\\tasetup.cmd"  {Need to stop TrustAgent service before running this}
4. About - Pops Intel TrustAgent Version Information
5. Exit - Exits the Tray Icon
